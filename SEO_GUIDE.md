# 🚀 Panduan SEO - Warung Badidud

## ✅ Optimasi yang Sudah Diterapkan

### 1. Meta Tags (Setiap Halaman)
- ✅ Title tag yang deskriptif dan mengandung keyword
- ✅ Meta description (150-160 karakter)
- ✅ Meta keywords
- ✅ Meta robots (index, follow)
- ✅ Canonical URLs
- ✅ Open Graph tags (Facebook/LinkedIn)
- ✅ Twitter Card tags

### 2. Structured Data (JSON-LD)
- ✅ **Homepage**: Restaurant schema
- ✅ **Menu**: Menu schema dengan MenuItem
- ✅ **Tentang**: Organization schema
- ✅ **Lokasi**: LocalBusiness dengan GeoCoordinates

### 3. File Teknis SEO
- ✅ `robots.txt` - Mengarahkan crawler
- ✅ `sitemap.xml` - Daftar halaman untuk Google
- ✅ Meta viewport untuk mobile-friendly

### 4. Performance
- ✅ Preload resource penting (logo, fonts)
- ✅ Lazy loading untuk images
- ✅ Reduced motion untuk accessibility
- ✅ Navigation preload saat hover

## 📋 Checklist SEO Tambahan

### Content SEO
- [ ] Tambahkan minimal 300-500 kata unik di setiap halaman
- [ ] Gunakan heading hierarchy (H1 → H2 → H3) dengan benar
- [ ] Tambahkan alt text deskriptif untuk semua gambar
- [ ] Internal linking antar halaman terkait

### Local SEO
- [ ] Daftar di **Google Business Profile** (gratis)
- [ ] Konsisten NAP (Name, Address, Phone) di semua platform
- [ ] Minta review dari pelanggan
- [ ] Tambahkan foto outlet secara berkala

### Technical SEO
- [ ] Submit sitemap ke **Google Search Console**
- [ ] Monitor Core Web Vitals di PageSpeed Insights
- [ ] Pastikan HTTPS sudah aktif
- [ ] Fix broken links (404 errors)

### Off-Page SEO
- [ ] Backlink dari website lokal (pemuda, kampus, dll)
- [ ] Social media presence (Instagram, Facebook, TikTok)
- [ ] Listing di Google Maps, TripAdvisor, Zomato
- [ ] Kolaborasi dengan food blogger/vlogger

## 🔧 Cara Submit ke Google

### 1. Google Search Console
1. Buka https://search.google.com/search-console
2. Login dengan akun Google
3. Tambahkan property: `warungbadidud.com`
4. Verifikasi kepemilikan (via DNS atau upload file HTML)
5. Submit sitemap: `https://warungbadidud.com/sitemap.xml`

### 2. Google Business Profile
1. Buka https://www.google.com/business/
2. Klik "Manage now"
3. Masukkan nama bisnis: "Warung Badidud"
4. Isi alamat lengkap dan jam operasional
5. Tambahkan foto menu dan outlet
6. Verifikasi via pos/kartu pos (biasanya 1-2 minggu)

### 3. Bing Webmaster Tools
1. Buka https://www.bing.com/webmasters
2. Sign in dengan Microsoft/Google account
3. Add site: `warungbadidud.com`
4. Submit sitemap

## 🚨 Common SEO Mistakes to Avoid

❌ **Keyword stuffing** - Jangan ulangi keyword berlebihan
❌ **Duplicate content** - Setiap halaman harus unik
❌ **Slow loading** - Optimize image size (< 200KB)
❌ **Not mobile-friendly** - Pastikan responsive di HP
❌ **Ignoring analytics** - Monitor traffic secara rutin
